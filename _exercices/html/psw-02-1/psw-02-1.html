PSW-02
Mission
Après avoir lu l'explication préliminaire ci-dessous, créer une page HTML affichant le message « Bonjour tout le monde ! » dans un conteneur div.
Si vous ne parvenez pas à réaliser la page après lecture de l'explication préliminaire, suivre le « Scénario d'aide » plus bas.
Explication préliminaire
Un page HTML — ou « page web » —, c'est d'abord un fichier portant l'extension .html ou .htm. Par exemple ma_page.html. Ce fichier contient le code HTML de la page.
On peut voir une page HTML comme un ensemble de boites imbriquées les unes dans les autres. On les appelle des « conteneurs ».
Un conteneur est définie par une balise (« tag » en anglais). Pour définir la boite, on a une balise d'ouverture (qui commence par « < ») et une balise de fermeture (qui commence par « </ ») :
  <boite>
 
    Contenu de la boite, du conteneur.
 
  </boite>
Ci-dessus, le mot boite est donné seulement pour l'exemple.
Une page web est une page HTML dont le conteneur principal est une balise html.
<html>
 
  Contenu de la page web
 
</html>
