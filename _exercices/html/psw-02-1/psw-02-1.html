Mission
Après avoir lu l'explication préliminaire ci-dessous, créer une page HTML affichant le message « Bonjour tout le monde ! » dans un conteneur div.
Si vous ne parvenez pas à réaliser la page après lecture de l'explication préliminaire, suivre le « Scénario d'aide » plus bas.
Explication préliminaire
Un page HTML, c'est d'abord un fichier portant l'extension `.html` ou `.htm.` Par exemple `ma_page.html`. Un fichier qui contient le code HTML de la page.
On peut voir une page HTML comme un ensemble de boites les unes dans les autres. On les appelle des « conteneurs ».
Un conteneur est définie par une balise (tag) d'ouverture et une balise de fermeture :
  <boite>
    Contenu du conteneur
  </boite>
(le mot `boite` peut être remplacé `div` ou `ul` ou `section`, ou plein d'autres choses)
Une page web est une page HTML dont le conteneur principal est une balise html.
